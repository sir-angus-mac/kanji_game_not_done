# 漢字学習ゲーム

このスマホゲームは、楽しく漢字を学習するために作られました。
ゲームのゴールは、画面上に漢字が積み上がってしまわないように、正しい言葉を組み合わせて漢字を「消していく」ことです！

> ⚠️ **お知らせ**: 現在は開発中のバージョン（未完成版）です。

## 🎮 操作方法

### ✍️ 文字を出現させる
画面上に文字を手書きし、下部の候補リストから正しい文字をタップすると、その文字が上からドロップします。

### ↕️ 移動とサイズ変更
まだ着地していない（落下中の）文字を長押しすると、自由に移動させることができます。
長押ししたまま**上**にドラッグすると文字が拡大し、**下**にドラッグすると縮小します。

### 📖 単語リストの確認
画面にある黒い漢字をダブルタップすると、その漢字を使った熟語や単語のリストが開きます。
リストの最上部にある文字をタップすると、その言葉のフリガナが表示されます。
